=== Description ===

This plugin allow you to put nice flash banner on your blog, banner will count number of days left
till Christmas. 

Plugin can show different messages:

1. "XX Days till Christmas" - Number of days till Christmas
2. "1 Day till Christmas" - When one day left till Christmas
3. "Less than a Day left till Christmas" - Message will be shown in 24st of December, when till christmas left less than a Day
4. "We wish you a Merry Christmas" - This message will be shown in 25st of December. 


== Installation ==

1. Upload plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `<?php christmassprite_countdown(); ?>` in your templates

You can also use Dutch version of this plugin, just replace christmassprite_countdown(); with christmassprite_countdown('ChristmasTree_Dutch');
Thanks to Bas Prinsen for translation.
